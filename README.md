# CutiaEngine
## Game Engine Cpp
<br>

### Depencies

```
$ sudo apt install cmake libsdl2-dev g++
```

[SDL install with cmake](https://trenki2.github.io/blog/2017/06/02/using-sdl2-with-cmake/)

### Run
```
$ make run
```
