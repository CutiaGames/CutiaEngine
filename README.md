### Install Depencies

#### Linux
```
$ sudo apt install cmake libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev libsdl2-mixer-dev g++
```

#### OSX with brew
```
$ brew install sdl2 sdl2_image sdl2_mixer sdl2_ttf
```

## Running the Engine

Running the project will create files in 'build' folder
### Run

```
$ make run
```

### Clean
```
$ make clean
```
